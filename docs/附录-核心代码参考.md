## B. VPN 连接管理器：VpnConnectionManager

TV 端需要去掉 Hilt 注解（`@Inject`、`@Singleton`），改为手动单例。

```kotlin
// 手机端原始代码（com.v2rayng.myvpn.vpn.VpnConnectionManager）
// TV 端复制后：去掉 @Inject/@Singleton，改为 object 或手动单例

package com.v2rayng.mytv.vpn

import android.content.Context
import android.content.Intent
import android.net.VpnService
import android.util.Log
import com.v2rayng.mytv.data.ServerDto
import kotlinx.coroutines.flow.MutableStateFlow
import kotlinx.coroutines.flow.StateFlow
import kotlinx.coroutines.flow.asStateFlow

enum class VpnState {
    DISCONNECTED, CONNECTING, CONNECTED, DISCONNECTING, ERROR
}

data class TrafficStats(
    val uploadSpeed: Long = 0,
    val downloadSpeed: Long = 0,
    val totalUpload: Long = 0,
    val totalDownload: Long = 0
)

class VpnConnectionManager(private val context: Context) {

    private val _vpnState = MutableStateFlow(VpnState.DISCONNECTED)
    val vpnState: StateFlow<VpnState> = _vpnState.asStateFlow()

    private val _trafficStats = MutableStateFlow(TrafficStats())
    val trafficStats: StateFlow<TrafficStats> = _trafficStats.asStateFlow()

    private var currentServer: ServerDto? = null

    fun needVpnPermission(): Boolean = VpnService.prepare(context) != null

    fun getVpnPermissionIntent(): Intent? = VpnService.prepare(context)

    fun startVpn(server: ServerDto, userUuid: String, proxyMode: String = "global") {
        currentServer = server
        _vpnState.value = VpnState.CONNECTING
        try {
            val config = V2RayConfig.generateConfig(server, userUuid, proxyMode)
            val intent = Intent(context, MyVpnService::class.java).apply {
                action = MyVpnService.ACTION_START
                putExtra(MyVpnService.EXTRA_CONFIG, config)
                putExtra(MyVpnService.EXTRA_SERVER_NAME, server.name)
                putExtra(MyVpnService.EXTRA_SERVER_HOST, server.host)
            }
            context.startForegroundService(intent)
        } catch (e: Exception) {
            Log.e("VpnConnectionManager", "Failed to start VPN: ${e.message}", e)
            _vpnState.value = VpnState.ERROR
        }
    }

    fun stopVpn() {
        _vpnState.value = VpnState.DISCONNECTING
        val intent = Intent(context, MyVpnService::class.java).apply {
            action = MyVpnService.ACTION_STOP
        }
        context.startService(intent)
    }

    fun updateState(state: VpnState) { _vpnState.value = state }
    fun updateTrafficStats(stats: TrafficStats) { _trafficStats.value = stats }
    fun resetTrafficStats() { _trafficStats.value = TrafficStats() }
    fun getCurrentServer(): ServerDto? = currentServer

    companion object {
        fun formatSpeed(bytesPerSecond: Long): Pair<String, String> = when {
            bytesPerSecond < 1024 -> Pair("%.0f".format(bytesPerSecond.toFloat()), "B/s")
            bytesPerSecond < 1024 * 1024 -> Pair("%.1f".format(bytesPerSecond / 1024f), "KB/s")
            bytesPerSecond < 1024L * 1024 * 1024 -> Pair("%.1f".format(bytesPerSecond / (1024f * 1024f)), "MB/s")
            else -> Pair("%.2f".format(bytesPerSecond / (1024f * 1024f * 1024f)), "GB/s")
        }

        fun formatBytes(bytes: Long): Pair<String, String> = when {
            bytes < 1024 -> Pair("$bytes", "B")
            bytes < 1024 * 1024 -> Pair("%.1f".format(bytes / 1024f), "KB")
            bytes < 1024L * 1024 * 1024 -> Pair("%.1f".format(bytes / (1024f * 1024f)), "MB")
            else -> Pair("%.2f".format(bytes / (1024f * 1024f * 1024f)), "GB")
        }
    }
}
```

## D. VPN 服务：MyVpnService

这是最核心的文件。TV 端直接复制，改包名即可。关键点：

- 继承 `VpnService()` 和 `CoreCallbackHandler`
- 使用 `libv2ray` 的 `Libv2ray.initCoreEnv()` 初始化 xray-core
- 使用 `Libv2ray.newCoreController(this)` 创建控制器
- `startLoop(config, tunFd)` 启动 xray-core
- `stopLoop()` 停止
- `queryStats("proxy", "uplink/downlink")` 查询流量
- TUN 接口配置：地址 `26.26.26.1/30`，路由 `0.0.0.0/0` + `::/0`
- DNS：`8.8.8.8` + `8.8.4.4`
- 排除自身包名防止路由循环

1. 包名 `com.v2rayng.myvpn` → `com.v2rayng.mytv`
2. `import com.v2rayng.myvpn.MainActivity` → `import com.v2rayng.mytv.MainActivity`
3. `import com.v2rayng.myvpn.R` → `import com.v2rayng.mytv.R`

## E. Tun2Socks（TUN 接口 ↔ SOCKS5 桥接）

纯 Kotlin 实现的 tun2socks，不依赖 native .so 文件。直接复制改包名。

**工作原理：**
1. 从 TUN fd 读取 IP 包
2. TCP 包：通过 SOCKS5 代理（127.0.0.1:10808）转发
3. UDP 包（DNS）：通过 protected socket 直接转发到 DNS 服务器
4. 响应包写回 TUN fd

**完整源码见手机端：** `E:\v2boradvpn\app\src\main\java\com\v2rayng\myvpn\vpn\Tun2Socks.kt`

TV 端只需改包名，无其他改动。

## G. libv2ray API 接口说明

`libv2ray.aar` 是 xray-core 的 gomobile 绑定，提供以下 Java/Kotlin API：

```kotlin
import libv2ray.CoreCallbackHandler
import libv2ray.CoreController
import libv2ray.Libv2ray

// 初始化核心环境（在 Application 或 Service 的 onCreate 中调用一次）
Libv2ray.initCoreEnv(filesDir: String, assetDir: String)

// 获取 xray-core 版本
Libv2ray.checkVersionX(): String

// 创建核心控制器（传入回调处理器）
Libv2ray.newCoreController(handler: CoreCallbackHandler): CoreController

// CoreController 方法：
controller.startLoop(config: String, tunFd: Int)  // 启动 xray-core
controller.stopLoop()                              // 停止
controller.isRunning: Boolean                      // 是否运行中
controller.queryStats(tag: String, direct: String): Long  // 查询流量统计
// tag = "proxy", direct = "uplink" 或 "downlink"

// CoreCallbackHandler 接口（MyVpnService 需要实现）：
interface CoreCallbackHandler {
    fun startup(): Long
    fun shutdown(): Long
    fun onEmitStatus(status: Long, msg: String?): Long
}

// 关键：必须在 Service.onCreate() 中调用
go.Seq.setContext(vpnService)  // 让 Go 层能调用 VpnService.protect()
```

**libv2ray.aar 位置：** 从手机端项目 `E:\v2boradvpn\app\libs\libv2ray.aar` 复制到 TV 端 `app/libs/`
